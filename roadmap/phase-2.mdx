## API & Integrations

### Developer Tools

<CodeGroup>
```typescript SDK Example
import { PrediqtSDK } from '@prediqt/sdk';

const sdk = new PrediqtSDK({
  network: 'mainnet',
  apiKey: 'your-api-key'
});

// Get market data
const market = await sdk.getMarket('bitcoin-50k-december');
const odds = market.getCurrentOdds();

// Place a bet
const bet = await sdk.placeBet({
  marketId: 'bitcoin-50k-december',
  position: 'YES',
  amount: 100,
  currency: 'PREDIQT'
});
```
